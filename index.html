HTML/CSS/JS プレゼン

HTML／CSS／JavaScriptを生成AIで
フルスクラッチして作るプレゼン資料

ChatGPTなどの生成AIを活用し、
Webベースのプレゼン資料を1つのHTMLファイルにまとめる方法を紹介します。

自己紹介

鉄道会社のIT部門で内製開発をしています。
takoyaki3.comという個人サイトなど趣味開発も好きです。

背景と目的

  • AIで自動化し、スライド作成の手間を削減
  • ブラウザだけで閲覧可能な配布しやすい資料
  • HTML1枚でコード共有も簡単

使用技術

  • HTML5 / CSS3 / Vanilla JavaScript
  • ChatGPT（生成AI）
  • Chart.js for グラフ

実装手順

  1. プロンプト設計で要件をAIに伝える
  2. AIからHTMLテンプレート生成
  3. CSS & JSを統合しデザイン調整
  4. 動作確認→ブラッシュアップ
