HTML／CSS／JavaScriptを生成AIでフルスクラッチして作るプレゼン資料

ChatGPTなどの生成AIを活用して、Webベースのプレゼン資料を1つのHTMLファイルだけで構築する方法を紹介します。

背景と目的

  • スライド作成をAIで自動化し、手間を削減
  • Web技術だけでブラウザ表示可能な資料を生成
  • コード共有が簡単：HTML1枚で配布

使用技術

  • HTML5 / CSS3 / JavaScript (Vanilla)
  • ChatGPT (生成AI)
  • 追加ライブラリ不要

実装手順

  1. プロンプト設計：目的・構成・要件をAIに明示
  2. AI出力のHTMLテンプレートを取得
  3. CSSとJSを同一ファイルに統合
  4. 動作確認＆ブラッシュアップ

コーディングデモ

以下はAIに依頼して得たサンプルコードです（抜粋）。

<button id="menu-btn">☰</button>

// JSでサイドバーをトグルする
const btn = document.querySelector('#menu-btn');
btn.addEventListener('click', () => {
  document.querySelector('#sidebar').classList.toggle('open');
});

応用アイデア

  • 社内ドキュメントの自動プレゼン化
  • APIリファレンスのWebスライド化
  • 教育教材の自動生成

まとめ・今後の展望

生成AIを活用すれば、コーディングの敷居を下げつつ、オリジナルのWebプレゼン資料を短時間で構築可能です。今後は以下を検討できます：

  • テーマカラーの自動最適化
  • Markdown → 自動HTML化パイプライン
  • リアルタイム共同編集機能の追加
